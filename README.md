# Usage

If you want the daily word, use:
 
```js
node wordle.js
```
 
If you want to see a word in the past / future, make sure you write the date in the format: [YYYY-MM-DD], example:
 
```js
node wordle.js 2023-09-09
```
 
If you want to see the next n set of words, use +n to see future ones, or -n to see ones that have past. Example showing the current word + next 6 words [next 7 words]:
 
```js
node wordle.js +7
```
 
Example showing the past 30 words [shows the current word + 29 previous words]:
 
```js
node wordle.js -30
```
